📘 Simple Student Record Management System (Python File Handling)
📌 Project Description

This is a simple Student Record Management System built using Python and File Handling.
The program allows users to:

➕ Add a new student record

📄 View all student records

❌ Exit the program

Student data is stored permanently inside a text file (Students.txt).

🛠 Features

Add student name, roll number, and marks

Store data in a text file using append mode

Read and display stored student records

Handle file not found errors

Simple menu-driven interface
